About WARCHIEF

WARCHIEF is your Warhammer 40,000 10th Edition companion. All data is stored in your browser's localStorage — it survives page reloads and browser restarts, but is cleared if you clear your browser data.

AI features (Datasheet Lookup & Photo Import) require an Anthropic API key. Add yours in Settings.

To host for your group: drag this HTML file onto netlify.com (free, no account needed). Each device gets its own independent data.
